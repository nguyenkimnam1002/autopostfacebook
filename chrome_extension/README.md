# Affiliate Hot Tool Connector

Chrome extension kết nối Chrome đang đăng nhập Shopee/Affiliate với tool localhost.

## Cài đặt

1. Mở Chrome.
2. Vào `chrome://extensions`.
3. Bật `Developer mode`.
4. Bấm `Load unpacked`.
5. Chọn thư mục:

```text
D:\WORK_2\hayho\affiliate_hot_tool\chrome_extension
```

Sau khi cài, mở:

```text
http://127.0.0.1:8001/affiliate_hot_tool
```

Tool sẽ ưu tiên dùng extension để gọi Shopee bằng phiên đăng nhập hiện tại của Chrome.
